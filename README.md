advancevim
==========
##Introduction
Vim and emacs is the most popular used editor in *inx editor
with the given plugins user can use vim more conveniently,we included the ctas,cscopes plugins .etc.
##Features
- ctags
- cscope
- TagList
- WinManager
- MiniBufExplorer
- ShowMarks
- EchoFunction
- Vim org mode
- calendar
- textmateOnly
- emmet-vim(https://github.com/mattn/emmet-vim)
##Getting Started
-Install ctags
  tar xf ctags-5.8.tar.gz
  cd ctags-5.8
  ./configure
  make
  sudo make install
-Install cscopes
  tar xf cscop-15.8a.tar.gz
  cd cscop-15.8a
  ./configure
  make
  sudo make install
